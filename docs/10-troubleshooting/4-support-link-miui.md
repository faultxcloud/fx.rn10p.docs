# Fix "Open support link" MIUI  
---

Setiap saya membuka link (contoh: youtube), selalu diarahkan ke browser tidak ke aplikasi YouTube.  
Saya sudah coba atur dari settings tetapi malah balik lagi ke semula. Akhirnya saya coba menggunakan aplikasi pihak ke tiga dan bisa juga akhirnya..  
Agar langsung bisa membuka ke aplikasi YouTube, berikut caranya:

> Oia bukan cuma aplikasi YouTube, tetapi aplikasi yang memang ada akses "Open by default".

- Download [Hidden Settings For MIUI](https://play.google.com/store/apps/details?id=com.ceyhan.sets)
- Open "Manage application"
- example, choose Youtube
- Open by default
- Add link
- Done
