# Rooting
Proses membuka akses penuh (superuser) pada sistem Android sehingga pengguna dapat mengubah, memodifikasi, atau menghapus komponen sistem yang sebelumnya tidak bisa diakses.  

Dengan rooting, pengguna memperoleh kontrol penuh atas perangkat—mirip seperti akses administrator di komputer.  

### ✨ Kelebihan Rooting
- Menghapus bloatware (aplikasi bawaan yang tidak bisa dihapus)
- Memasang aplikasi khusus yang butuh akses root
- Modifikasi sistem (tema, tweak, performance)
- Memasang modul seperti Magisk, Dolby, Viper4Android, dll.

### ⚠️ Risiko Rooting
- Potensi keamanan berkurang
- Bisa menyebabkan bootloop jika salah modifikasi
- Garansi dapat batal
- Beberapa aplikasi (bank/e-wallet) mendeteksi perangkat root

#### 📝 Kesimpulan
Rooting memberi akses penuh ke sistem Android, memungkinkan kustomisasi mendalam, tetapi juga membawa risiko keamanan dan stabilitas. Gunakan hanya jika memahami konsekuensinya.
